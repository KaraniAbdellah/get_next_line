# get_next_line

The `get_next_line` function reads the next line from a file.

| Function Prototype                              | Description                                                                                   |
|-------------------------------------------------|-----------------------------------------------------------------------------------------------|
| `char *get_next_line(FILE *p_file, int *position);` | Gets the next line from a file and returns it as a string. |
